# E-Commerce Sales Analytics Dashboard

## Project Overview
This project is an interactive E-Commerce Sales Analytics Dashboard developed using Python in Visual Studio Code (VS Code). It analyzes e-commerce sales data to provide insights into sales performance, top-selling products, category-wise sales, regional trends, and customer purchasing patterns.

## Tools & Technologies
- Python
- Visual Studio Code (VS Code)
- Pandas
- NumPy
- Plotly
- Streamlit
- Matplotlib
- Seaborn

## Features
- Sales Performance Analysis
- Category-wise Sales Insights
- Region-wise Sales Analysis
- Top Selling Products
- Interactive Dashboard
- Data Visualization and Reporting
